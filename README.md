# TP-Fallas-1
Trabajo Práctico Final. Análisis y Detección de Fallas. Universidad de Buenos Aires

### Requerimientos:
 * Python 3.6+
 * Biblioteca <i> Experta </i>
 * Flask

## Cómo instalar <i> Experta </i>:
```
pip install experta
```

## Cómo instalar Flask:
```
pip install Flask
```
## Cómo levantar la aplicación:
```
flask run --reload
```
y entrar a http://127.0.0.1:5000/
